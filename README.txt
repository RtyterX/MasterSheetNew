==============================
       MASTERSHEET NEW*
==============================

Descrição
----------
MasterSheetNew é uma aplicação designada para subistituir a MasterSheet7.5,
um gerador de Scripts para routers CPEs (Diversos Fabricantes) com base em Excel.

Foi projetada para auxiliar o usuario na configuração, coleta de logs e até mesmo
no preenchimentos de Tickets de atendimento. Unindo outras ferramentas de 
formatação de Texto, ela unifica diversas ferramentas em um só canal.


Features
---------
- Criação de Clientes e Padronização de Processos
- Configuração de CPEs Cisco, HPE, Huawei e Fortigate (Dados)
- Configuração de CPEs para circuitos de Voz: Cisco, HPE, Aligera e Digistar.
- Coleta de Logs e Debug para Circuitos de Voz
- Coleta de Logs para CPEs Cisco, HPE, Huawei e Fortigate
- Coleta de Logs para Backbones Cisco e Nokia
- Abertura de Tickets de Suporte
- Gerador de Log de Pendencia 
- Calculadora de Bloco de IPv4


Requerimentos
--------------
Náo há requerimentos. A ideia é que o programa seja executado em qualquer computador
que possua Windows. Sem necessidade de conexäo com a internet.


Instalação
-----------
1. Baixar o arquivo zipado
2. Extrair os arquivos para a pasta desejada
3. Execute o Programa
